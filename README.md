# FastAPI Support Tickets

REST API для тикетов службы поддержки на FastAPI + MySQL.

## Возможности

- Создание тикетов с прикреплением файлов
- Статусы: `new`, `in_progress`, `answered`, `closed`, `deferred`
- Комментарии к тикетам с файлами
- Фильтрация тикетов по дате и статусу (только для админа)
- UUID для безопасного доступа
- Два типа API-ключей: пользовательский и административный

## Установка

```bash
pip install -r requirements.txt
cp .env.example .env
# Отредактируйте .env — укажите параметры БД и API-ключи
uvicorn main:app --reload
```

## Переменные окружения (.env)

| Переменная      | Описание                          |
|-----------------|-----------------------------------|
| `DATABASE_URL`  | URL подключения к MySQL           |
| `API_KEY_USER`  | API-ключ для пользователей        |
| `API_KEY_ADMIN` | API-ключ для администраторов      |
| `UPLOAD_DIR`    | Папка для загружаемых файлов      |

Создать API ключ:
```bash
python -c "import uuid; print(uuid.uuid4())"
```

## Аутентификация

Передавайте ключ в заголовке `X-API-Key`.

- `API_KEY_USER` — доступ к пользовательским эндпоинтам
- `API_KEY_ADMIN` — доступ ко всем эндпоинтам, включая административные

## API эндпоинты

### Пользовательские (API_KEY_USER или API_KEY_ADMIN)

| Метод  | URL                                    | Описание                          |
|--------|----------------------------------------|-----------------------------------|
| POST   | `/tickets`                             | Создать тикет (multipart/form)    |
| GET    | `/tickets/{uuid}`                      | Получить тикет с комментариями    |
| POST   | `/tickets/{uuid}/comments`             | Добавить комментарий              |
| PATCH  | `/tickets/{uuid}/close`                | Завершить тикет                   |

### Административные (только API_KEY_ADMIN)

| Метод  | URL                                    | Описание                          |
|--------|----------------------------------------|-----------------------------------|
| GET    | `/admin/tickets`                       | Список тикетов (фильтры: status, date_from, date_to) |
| GET    | `/admin/tickets/{uuid}`                | Тикет с комментариями             |
| PATCH  | `/admin/tickets/{uuid}/status`         | Изменить статус тикета            |
